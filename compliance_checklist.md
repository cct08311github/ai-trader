## 審計軌跡記錄格式

### 合規狀態變更記錄
每次合規事項狀態變更時，應記錄以下資訊：

| 欄位 | 說明 |
|------|------|
| **事項編號** | 對應合規事項編號 (如：1.1) |
| **變更前狀態** | 未開始 / 進行中 / 已完成 |
| **變更後狀態** | 未開始 / 進行中 / 已完成 |
| **變更日期** | YYYY-MM-DD HH:MM:SS |
| **變更人員** | 執行變更的人員/系統 |
| **變更理由** | 簡述變更原因 |
| **證據連結** | 相關證據檔案路徑或識別碼 |
| **簽核人員** | 如有簽核，記錄簽核人員 |

### 合規審計查詢
系統應提供以下審計查詢功能：
- 依日期範圍查詢合規狀態變更
- 依事項編號查詢完整歷史記錄
- 合規完成度統計報表
- 逾期未完成事項警示
